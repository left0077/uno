## 快速测试步骤

1. **打开浏览器**: 访问 http://localhost:3000
2. **输入昵称**: 如 "测试玩家1"
3. **创建房间**: 点击"创建房间"按钮
4. **复制房间号**: 点击"复制房间号"
5. **添加AI**: 点击"添加AI"选择难度
6. **开始游戏**: 点击"开始游戏"

---

## 多窗口测试

可以打开多个浏览器标签页测试多人对战：

```bash
# 标签页1 - 房主
http://localhost:3000
昵称：玩家1
创建房间

# 标签页2 - 加入者  
http://localhost:3000
昵称：玩家2
加入房间（输入标签页1的房间号）
```
